# vue-move-number

基于vue开发的数字滚动组件

## Build Setup

``` bash
# install dependencies
npm install --save vue-move-number
```
## 使用方法

### 引入
```
import MoveNumber from "vue-move-number"
Vue.use(MoveNumber);
```

```
<move-number :number=num :width=width :height="height"></move-number>   //num传入的数字， width和height是每个数字的宽高

```
